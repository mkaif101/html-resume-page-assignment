HTML Resume Page Assignment

This project is a single-page resume website created using only HTML, based on the provided reference layout.
No CSS or JavaScript has been used in this assignment.

The resume includes sections such as header, about, skills, experience, projects, education, and contact details.
Dummy data has been used to represent the content.

To run the project:
1. Download or clone the repository
2. Open the index.html file in any web browser

Screenshots of the final output are included in the repository.
